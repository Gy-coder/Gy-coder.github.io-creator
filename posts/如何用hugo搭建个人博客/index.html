如何用Hugo搭建个人博客

November 10, 2019

安装

  1. 在Mac上 在Mac上安装hugo非常方便，仅需要两个命令就可以完成

    brew install hugo

    验证hugo的安装是否成功

    hugo version

  2. 在Windows上

    • 去Hugo release页面下载hugo_xxx_Windows-64bit.zip
    • 解压，把hugo.exe放到D:\Software\hugo\hugo.exe
    • 把D:\Software\hugo\加到PATH中

    • 重启终端，运行hugo version查看版本

      快速搭建博客

  3. 进入Hugo官网，点击Quick Start

  4. 从Step 2到Step 7按照网页上的内容依次操作

  5. 操作完毕后会得到一个public目录，这就是我们的博客站点

  6. 创建.gitignore文件，在文件内写入/public/
    (注：讲public与hugo的内容分离，上传到两个分支)

  7. cd public(进入public目录中) —>git init->git add . ->git commit (注：将public上传到本地的git仓库)

  8. 在GitHub中创建一个仓库，名为(用户名).github.io

例如：Gy-coder.github.io 7. git remote add origin (你的SSH地址)

git push -u origin master

(注：确保cd的是public，因为只上传public到.github,io仓库中) 8. 点击 setting，找到GitHub Pages，然后点击链接就可以看到blog的内容了
